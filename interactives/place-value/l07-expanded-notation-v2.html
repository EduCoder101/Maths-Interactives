Learning Intention and Success Criteria

Learning Intention

We are learning to partition numbers into expanded notation using place value.

Success Criteria

  • I can write a number in expanded form using addition.
  • I can identify the value each digit contributes to the total.
  • I can convert expanded notation back into a standard numeral.
  • I can partition numbers in non-standard ways using place value.

Why Is Expanded Notation Useful?

It shows you exactly what each digit is worth.

It helps with mental addition and subtraction strategies.

It is the foundation for understanding how algorithms work.

Big Idea

When you understand the parts, you understand the whole.

Expanding 46,835

46,835
Step 1The 4 is in the ten-thousands place: 40,000
Step 2The 6 is in the thousands place: + 6,000
Step 3The 8 is in the hundreds place: + 800
Step 4The 3 is in the tens place: + 30
Step 5The 5 is in the ones place: + 5
Step 6Put it all together: 46,835 = 40,000 + 6,000 + 800 + 30 + 5
Key Idea

Each part shows the value of one digit. Added together, they make the original number.

Go Deeper: What Can We Discover?

Investigation thread: What can we discover about different ways to partition numbers?

Specialising
How many different ways can you partition 2,400 into exactly two parts where both parts are multiples of 100? List as many as you can.
Start systematically. What if the first part is 100? What if it's 200?
A mathematician's response

I need two multiples of 100 that add to 2,400. Let me start with the smallest: 100 + 2,300 = 2,400. Then 200 + 2,200. Then 300 + 2,100. I see a pattern forming. 400 + 2,000. 500 + 1,900. 600 + 1,800. 700 + 1,700. 800 + 1,600. 900 + 1,500. 1,000 + 1,400. 1,100 + 1,300. 1,200 + 1,200. After that, they start repeating. So there are 12 unique pairs if we don't count reversed pairs separately.

Mathematician's thinking: You are specialising. That means you are trying specific examples to gather evidence, looking for a pattern before you make any claims.
Conjecturing
'When a number has a zero in it, the expanded form always has fewer terms than the number has digits.' Is this always true, sometimes true, or never true?
Test with numbers that have 1 zero, 2 zeros, and 3 zeros.
A mathematician's response

Let me test this systematically. Test 1: 4,032 has 4 digits and 1 zero. Expanded: 4,000 + 30 + 2 = 3 terms. Fewer? Yes (3 < 4). Test 2: 5,005 has 4 digits and 2 zeros. Expanded: 5,000 + 5 = 2 terms. Fewer? Yes (2 < 4). Test 3: 10,000 has 5 digits and 4 zeros. Expanded: 10,000 = 1 term. Fewer? Yes (1 < 5). But wait, what about 10 (2 digits, 1 zero)? Expanded: 10 = 1 term. 1 < 2, so yes. What about 101 (3 digits, 1 zero)? Expanded: 100 + 1 = 2 terms. 2 < 3, yes. I think this is ALWAYS TRUE. Each zero removes one term from the expanded form.

Mathematician's thinking: You are conjecturing. That means you are making a claim that you believe is true, based on your evidence, but you know it still needs to be proved.
Justifying
A student says: '45,000 + 3,000 + 200 = 48,200, so 45,000 must be a valid part of the expanded notation.' Is this student's reasoning correct? What is the difference between expanded notation and non-standard partitioning?
Think about what 'expanded notation' specifically means versus just breaking a number into parts.
A mathematician's response

The student is right that 45,000 + 3,000 + 200 = 48,200. But is 45,000 a valid term in expanded notation? I need to think about what expanded notation means. In standard expanded notation, each term represents exactly one digit's value. 48,200 = 40,000 + 8,000 + 200. Each term is a single non-zero digit times a power of 10. But 45,000 is not a single digit times a power of 10: it's 4 ten-thousands and 5 thousands combined into one term. So the student's partition is valid (it does add to 48,200) but it's a NON-STANDARD partition, not expanded notation. The difference: expanded notation splits a number so each term shows exactly one digit's place value. Non-standard partitioning can split any way that adds up. Both are correct, but they serve different purposes.

Mathematician's thinking: You are justifying. That means you are not just checking whether an answer is right, you are examining whether the reason given is correct. A right answer with a wrong reason is still a problem in mathematics.
