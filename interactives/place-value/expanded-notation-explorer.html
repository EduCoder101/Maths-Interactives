Learning Intention & Success Criteria ▼

Learning Intention

We are learning to explore how numbers can be broken apart using expanded notation and how each digit contributes to the total value.

Success Criteria

  • I can write a number in expanded notation showing the value of each digit
  • I can identify the value of each digit in a multi-digit number
  • I can switch between standard form and expanded form
