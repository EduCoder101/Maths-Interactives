The Value Detective

Find the value of each digit

Learning Intention & Success Criteria ▼

Learning Intention

We are learning to identify the value of a digit based on its position in a number.

Success Criteria

  • I can determine the place value column a digit sits in
  • I can state the actual value of a digit in a number (e.g. the 4 in 5,420 is worth 400)
  • I can explain why the same digit can have different values in different positions

How to Be a Value Detective

Each digit in a number has a different value depending on its position. Your job is to find the actual value of the underlined digit!

The Detective's Rule

The position of a digit determines its value. The same digit "4" can be worth 4, 40, 400, 4,000, or even 4,000,000 depending on where it sits!

Example: In 5,420, the underlined 4 is in the hundreds place.
So its value is 400 (not just 4!)

Worked Example

Let's work through a question together, step by step.

What is the value of the underlined digit in 38,205 ?

Step 1: Find the underlined digit. It is 8.
Step 2: Work out which column it sits in. Counting from the right: ones, tens, hundreds, thousands.
Step 3: Multiply the digit by its column value: 8 × 1,000 = 8,000
Answer: The value of the underlined 8 is 8,000

Go Deeper: Place Value Investigations

You've mastered the skills. Now think like a mathematician. These tasks are not about right or wrong answers. They are about exploring, reasoning, and convincing others.

Specialising
In the number 52,347, the digit 5 is worth 50,000.

What if the 5 moved one column to the right? What would it be worth now? What if it moved two columns to the right?

Try this with at least 3 different starting numbers and digits.
Each time a digit moves one column to the right, what happens to its value? What about moving to the left?
A mathematician's response

In 52,347, the 5 is in the ten thousands column, worth 50,000.

Move it one column right (thousands): now worth 5,000. That is 10 times smaller.

Move it two columns right (hundreds): now worth 500. That is 100 times smaller than where it started.

I tried 1,836. The 8 in the hundreds column is worth 800. Move it left to thousands: 8,000. That is 10 times bigger.

Each column to the right divides the value by 10. Each column to the left multiplies the value by 10. The digit stays the same, but its position decides its value.

Mathematician's thinking: You are specialising. That means working through specific examples carefully to notice something. The best discoveries come from trying lots of cases and asking "what changed?"
Conjecturing
A 6-digit number contains exactly one 7, and that 7 has a value of 70,000.

How many different 6-digit numbers could it be? Can you work it out without listing every single one?
Think about which column the 7 must be in. Then think about what choices you have for each of the other five digits.
A mathematician's response

For the 7 to be worth 70,000, it must sit in the ten thousands column. In a 6-digit number, that is the second position: _ 7 _ , _ _ _

The first digit (hundred thousands) can be 1, 2, 3, 4, 5, 6, 8, or 9. It cannot be 0 (or it would not be a 6-digit number) and it cannot be 7 (because there is only one 7 in the number). That gives 8 choices.

Each of the remaining 4 digits (thousands, hundreds, tens, ones) can be 0, 1, 2, 3, 4, 5, 6, 8, or 9. They cannot be 7 (only one 7 allowed). That gives 9 choices each.

Total: 8 x 9 x 9 x 9 x 9 = 52,488 numbers. You do not need to list them all. The shortcut is to count the choices for each position and multiply!

Mathematician's thinking: You are conjecturing. That means making a testable claim about how many possibilities there are. Mathematicians look for shortcuts and patterns rather than listing every case by hand.
Generalising
In 3,482, the 3 is worth 3,000.
In 34,820, the 3 is worth 30,000.

The second number is the first number multiplied by 10. The value of the 3 also got 10 times bigger.

Does this always happen? If you multiply a number by 10, does every digit's value get 10 times bigger? What about multiplying by 100?
Try it with at least 3 different numbers. Can you write a rule that connects multiplying by 10 to what happens in the place value columns?
A mathematician's response

256: the 2 is worth 200. Multiply by 10: 2,560. Now the 2 is worth 2,000. That is 10 times bigger.

4,091: the 9 is worth 90. Multiply by 10: 40,910. Now the 9 is worth 900. That is 10 times bigger.

73 x 100 = 7,300. The 7 went from 70 to 7,000. That is 100 times bigger.

My rule: When you multiply a number by 10, every digit moves one column to the left, so every digit's value gets 10 times bigger. Multiplying by 100 moves every digit two columns left, making each value 100 times bigger. This always works because our place value system is built on powers of 10.

Mathematician's thinking: You are generalising. That means looking across lots of examples to find a rule that always works. You are also making a connection between two ideas: place value and multiplication.
Justifying
Priya says: "In the number 848,484, all the 4s have the same value because they are the same digit."

Is Priya right or wrong? Write a clear explanation that would convince a younger student why.
Can you use a place value chart to help explain? What would you tell Priya about the difference between a digit and its value?
A mathematician's response

Priya is wrong. The digit is the same (4), but the value depends on which column it sits in.

In 848,484:

The first 4 is in the ten thousands column, so it is worth 40,000.
The second 4 is in the hundreds column, so it is worth 400.
The third 4 is in the ones column, so it is worth 4.

A digit is just a symbol (0 to 9). Its value is what that symbol is actually worth, and that changes depending on its position. Think of it like a 1p coin and a £1 coin. They both have "1" on them, but they are not worth the same amount. Position matters.

Mathematician's thinking: You are justifying. That means explaining why something is true or false, not just saying the answer. A strong justification walks through the reasoning step by step so that someone else could follow it.
