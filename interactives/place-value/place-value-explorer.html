Learning Intention & Success Criteria ▼

Learning Intention

We are learning to compare the value of digits in different place value positions and describe the multiplicative relationship between them.

Success Criteria

  • I can identify which digit has a greater value based on its position
  • I can explain how many times larger or smaller one digit is compared to another
  • I can use the pattern of ×10 for each place value shift to reason about digit relationships

Comparing the Same Digit

When the same digit appears in two different places, we can work out how many times larger one is by counting the jumps between them.

300
3 in Hundreds
vs
30
3 in Tens
1 jump apart = ×10 larger

So the 3 in 300 is 10 times larger than the 3 in 30.

3
Counting the Jumps

Each jump to the left multiplies the value by 10:

1 jump×10
2 jumps×100
3 jumps×1,000
4 jumps×10,000
5 jumps×100,000

Try this: if a digit is in the tens place and another is in the hundreds place, how many jumps apart are they?

Answer: 1 jump = ×10 larger
