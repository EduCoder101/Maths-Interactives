Learning Intention and Success Criteria

Learning Intention

We are learning that the place value system extends beyond the ones place into tenths, hundredths, and thousandths.

Success Criteria

  • I can explain that each decimal place is ten times smaller than the place before it.
  • I can name the tenths, hundredths, and thousandths columns.
  • I can identify the value of each digit in a decimal number.
  • I can read a decimal by naming the value of each place.

Card 2 of 10

Zooming Into the Number Line

What if we zoom in between 0 and 1? What do we find?

Step 1Zoom into 0 to 1: We find 10 equal parts: 0.1, 0.2, 0.3... These are tenths.
Step 2Zoom into 0.1 to 0.2: We find 10 more equal parts: 0.11, 0.12, 0.13... These are hundredths.
Step 3Zoom into 0.11 to 0.12: We could find 10 more equal parts to show thousandths.
Key Pattern

The number system never ends. We can always zoom in further.

Go Deeper: Decimal Place Value Investigations

These tasks will push your thinking beyond finding answers. You will explore, make conjectures, and reason like a mathematician.

Specialising
Start with the number 5. Divide by 10 five times. Write each result. What do you notice about where the 5 moves?
Track the 5's position in a place value chart each time.
A mathematician's response

5 ÷ 10 = 0.5. 0.5 ÷ 10 = 0.05. 0.05 ÷ 10 = 0.005. 0.005 ÷ 10 = 0.0005. 0.0005 ÷ 10 = 0.00005.

The 5 moves one column to the right each time. First in ones, then tenths, hundredths, thousandths, ten-thousandths, hundred-thousandths. The pattern of ÷10 sends each digit one place to the right, making it ten times smaller each time.

Mathematician's thinking: You are specialising. That means you are trying specific examples to gather evidence, looking for a pattern before you make any claims.
Conjecturing
"Adding a zero after the last decimal digit never changes the value of a number." Always, sometimes, or never true?
Test with 0.5, 0.50, 0.500. Then test with 5, 50, 500.
A mathematician's response

0.5 = 0.50 = 0.500. Adding zeros after the last decimal digit does not change the value.

But wait: 5 and 50 and 500 are very different! Adding a zero AFTER a whole number makes it 10 times bigger. The key difference is whether the zero is after a decimal digit or after a whole number.

For decimals: ALWAYS TRUE. Adding trailing zeros after the last decimal digit does not change the value. But I was careful to note this only works after the decimal point.

Mathematician's thinking: You are conjecturing. That means you are making a claim that you believe is true, based on your evidence, but you know it still needs to be proved.
Generalising
Write a rule explaining how you know how much a digit is worth based only on its position after the decimal point. Can you express this using powers of 10?
The first place after the decimal is tenths. What fraction is that? What about the second place?
A mathematician's response

1st place after decimal: tenths = 1/10 = 1/(101). 2nd place: hundredths = 1/100 = 1/(102). 3rd place: thousandths = 1/1000 = 1/(103). 4th place: ten-thousandths = 1/10000 = 1/(104).

My rule: the nth place after the decimal point has a value of 1/(10n). So a digit d in the nth place is worth d/(10n). This connects to whole numbers where the nth place before the decimal has value 10(n-1).

Mathematician's thinking: You are generalising. That means you are taking a pattern you have found and writing a rule or formula that works for all cases, not just specific examples.
