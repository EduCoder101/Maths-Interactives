Lesson 12: Estimation, strategy selection, error analysis, and equivalent division

Learning Intention & Success Criteria ▼

Learning Intention

We are checking how well we can select efficient strategies, estimate to check reasonableness, identify errors, and apply equivalent division.

Success Criteria

  • I can choose the most efficient strategy for a given calculation.
  • I can use a Radar estimate to check any multiplication or division answer.
  • I can identify an error in a completed algorithm and explain the misconception.
  • I can find the missing number in an equivalent division.
  • I can solve rate problems using multiplication and division together.
