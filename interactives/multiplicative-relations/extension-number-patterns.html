Patterns in multiples

When you list the multiples of a number, you get a pattern with a constant difference between each term.

Example: multiples of 6

6, 12, 18, 24, 30, 36, ...

Each number is 6 more than the one before. The difference between consecutive terms is always 6.

You can also describe each term using its position: the 1st term is 6 x 1 = 6, the 2nd term is 6 x 2 = 12, and so on. The rule is: term = 6 x position.

Tables of values

A table of values shows the relationship between an input and an output. The rule connects them.

InputOutput
14
27
310
4?

Look at how the output changes. Each time the input goes up by 1, the output goes up by 3.

The constant difference is 3, so the rule involves multiplying by 3. But 3 x 1 = 3, not 4. There is an extra 1 each time.

The rule is: output = 3 x input + 1.

Check: 3 x 4 + 1 = 13. So the missing value is 13.

Strategy

First find the constant difference (how much the output changes each step). Then check if the rule needs a "plus" or "minus" adjustment by testing your rule on the first input-output pair.

These tasks push your thinking beyond completing patterns. You will create your own rules, test conjectures, and reason about what makes a pattern interesting.

Specialising
Create three different rules that all produce 20 as the output when the input is 4. Write out the first six values for each rule. What do the three rules have in common? Where do they differ?
Try to find rules with different constant differences.
A mathematician's response

Rule A: 5 x input. Outputs: 5, 10, 15, 20, 25, 30. Constant difference: 5.

Rule B: 3 x input + 8. Outputs: 11, 14, 17, 20, 23, 26. Constant difference: 3.

Rule C: 7 x input - 8. Outputs: -1, 6, 13, 20, 27, 34. Constant difference: 7.

All three rules pass through 20 at input 4, but they have different starting values and different rates of growth. The constant difference is the number you multiply by.

Mathematician's thinking: You are specialising. That means you are creating specific examples to investigate a question.
Conjecturing
Look at the outputs from a two-step rule (like 3 x input + 1). Is the constant difference always equal to the multiplier? Test this with at least three different rules. Make a conjecture about why.
What happens to the constant difference if you change the "plus" part?
A mathematician's response

4 x input + 3: outputs 7, 11, 15, 19. Difference: 4.

4 x input + 100: outputs 104, 108, 112, 116. Difference: still 4.

Changing the "plus" part shifts all values up or down, but the gap between consecutive terms stays the same. My conjecture: the constant difference is always equal to the multiplier, no matter what you add or subtract.

Mathematician's thinking: You are conjecturing. That means you are making a claim based on evidence, ready to be tested further.
Justifying
Explain why the constant difference is always equal to the multiplier. Use an example to support your argument. Could you convince a classmate who disagrees?
Think about what happens when you calculate the difference between two consecutive outputs.
A mathematician's response

If the rule is "multiply by 3, then add 1", the output for input 5 is 3 x 5 + 1 = 16, and for input 6 it is 3 x 6 + 1 = 19.

The difference is 19 - 16 = 3. The "+1" part appears in both outputs and cancels out when you subtract. Only the multiplied part matters. Since the input increased by 1, and you multiply by 3, the output increases by 3 x 1 = 3.

This will always work, because the added constant never changes, so it always cancels. That is why the constant difference depends only on the multiplier.

Mathematician's thinking: You are justifying. That means you are explaining not just what is true, but why it must be true.
