📊 Link your question bank to a Google Sheet

  1. Create a new Google Sheet (or use an existing one). When you save, questions are stored in separate tabs by year group: Year 1, Year 2, Year 3, Year 4, Year 5, Year 6. The script creates these tabs automatically.
  2. In the sheet menu: Extensions → Apps Script. Delete any sample code and paste in the script from the file DailyTen-SheetSync.gs in this project (see the HTML - INTERACTIVES folder). Save the project.
  3. In Apps Script: Deploy → New deployment. Choose type Web app. Set "Execute as" to Me, "Who has access" to Anyone. Deploy and copy the Web app URL.
  4. Paste that URL into the "Backup to Google Sheet" box above (it should end with /exec). Use Save to Sheet to upload your bank, and Load from Sheet to restore it.
  5. Important: Open this Daily Ten page from a web server (e.g. your school intranet or HTTPS), not by double‑clicking the HTML file. Browsers block requests to Google when the page is opened as a local file.
